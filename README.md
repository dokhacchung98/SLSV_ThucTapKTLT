# SLSV_ThucTapKTLT

## Ứng dụng quản lý sinh viên

**Các chức năng chính**
- Thêm sinh viên
- Sửa sinh viên
- Xóa sinh viên
- In danh sách
- Sắp xếp (Sắp xếp trộn, chèn, nhanh, .., tăng dần hay giảm dần)
- Tìm kiếm sinh viên
- Thống kê (Theo lớp, theo điểm)
